# Roadmap and Milestones

## M1 - 最小闭环

- ingest + 在线 ASR + deliverables（转译/摘要）
- 多项目并行、实时日志、进度、cancel

## M2 - 关键帧能力

- stable-frame（先全帧）
- diff 曲线与关键帧预览

## M3 - 图文交错

- OCR-only + timeline + illustrated notes

## M4 - 复杂视频适配

- ROI stable、聚类代表帧
- ASR 二次校正与策略兜底

## Exit Criteria

- 每个阶段有可验证产物
- 所有控制命令（pause/resume/cancel/delete）具备可预期语义
